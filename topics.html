⛏️ Bitcoin Mining

Bitcoin mining is the process by which new Bitcoin is created and transactions are confirmed on the network.

How It Works

  • Miners compete to solve complex mathematical puzzles
  • The first miner to solve the puzzle gets to add the next block
  • They receive newly minted Bitcoin as a reward
  • This process secures the network and confirms transactions

Key Concepts

  • Proof of Work: Requires computational effort to validate transactions
  • Hash Rate: Total computing power on the network
  • Difficulty: Adjusts every 2016 blocks to maintain 10-min block times
  • Block Reward: Currently 3.125 BTC per block (as of 2024)

💳 Bitcoin Wallets

A Bitcoin wallet is software that allows you to send and receive BTC.

Types of Wallets

  • Hot Wallets: Connected to the internet (mobile, desktop)
  • Cold Wallets: Offline storage (hardware, paper)
  • Hardware Wallets: Dedicated devices for maximum security
  • Custodial: Third-party holds your keys (exchanges)

Important Terms

  • Private Key: Secret that controls your Bitcoin
  • Public Key: Shareable address to receive Bitcoin
  • Seed Phrase: 12-24 word backup of your wallet
  • Multi-sig: Requires multiple keys to authorize transactions

⚡ Lightning Network

The Lightning Network is Bitcoin's second-layer solution for instant, low-cost transactions.

Why Lightning?

  • Enables millions of transactions per second
  • Near-instant confirmation times
  • Extremely low fees (fraction of a cent)
  • Perfect for micropayments and daily transactions

How It Works

  • Create a payment channel between two parties
  • Make unlimited transactions off-chain
  • Only the opening and closing transactions go on main chain
  • Channels can be routed through intermediate nodes

✂️ Bitcoin Halving

The halving is a programmed event that cuts the block reward in half.

Schedule

  • Occurs every 210,000 blocks (~4 years)
  • 2009-2012: 50 BTC per block
  • 2012-2016: 25 BTC per block
  • 2016-2020: 12.5 BTC per block
  • 2020-2024: 6.25 BTC per block
  • 2024-2028: 3.125 BTC per block

Why It Matters

  • Creates predictable, diminishing supply
  • Historically followed by price increases
  • Ensures total supply never exceeds 21 million
  • Makes Bitcoin increasingly scarce over time

🔒 Bitcoin Privacy

Bitcoin offers pseudonymous transactions, not complete anonymity.

Understanding Privacy

  • All transactions are publicly visible on the blockchain
  • Addresses are pseudonymous (not tied to identity by default)
  • Chain analysis can link addresses to identities
  • Reusing addresses reduces privacy

Privacy Tools

  • CoinJoin: Mix transactions with other users
  • Lightning Network: Off-chain transactions
  • New Addresses: Generate fresh address for each transaction
  • Hardware Wallets: Keep private keys offline

🌐 Bitcoin Nodes

A node is a computer that runs Bitcoin software and maintains the blockchain.

Why Run a Node?

  • Verify your own transactions without trusting others
  • Contribute to network decentralization
  • Full control over your financial sovereignty
  • Privacy: don't leak addresses to third parties

Types of Nodes

  • Full Node: Downloads and verifies entire blockchain
  • Pruned Node: Stores only recent blocks
  • Light Node: Simplified payment verification
  • Lightning Node: Runs Lightning channels
